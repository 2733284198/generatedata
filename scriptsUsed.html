Scripts / Tools Used

The Data Generator is built on top of the work of a LOT of people. A big thanks to the everybody involved in the following projects: you all rock. I would be proud if any of you wanted to date my sister.

  • jQuery
  • jQuery UI
  • RequireJS
  • CodeMirror
  • Smarty PHP
  • Chosen - select box enhancement script
  • jQuery JSON plugin
  • Bootstrap - used for this documentation
  • Sass - Syntactically Awesome Stylesheets
  • Spinners, Canvas library - this is a fork of mine by Nick Stakenburg to allow for a few extra features
  • ExplorerCanvas - used as a fallback for IE for Canvas
  • La Glanz Studio - spiffy dice icon (main logo)
  • Nice bug icon

Books & Other Resources

A big thanks to Addy Osmani for his brilliant article Patterns For Large-Scale JavaScript Application Architecture. The client-side code architecture for the Data Generator employs a simplified form of it.

The following books have also been extremely useful. All JS, you'll note. That's where the action's at.

  • JavaScript Patterns - Stoyan Stefanov
  • Learning JavaScript Design Patterns - Addy Osmani
  • JavaScript Web Applications - Alex MacCaw
  • High Performance JavaScript - Nicolas Zakas
